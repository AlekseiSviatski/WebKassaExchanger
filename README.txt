Инструкция по настройке модуля:

1. В личном кабинете настоящей цифровой узнать номера всех программных касс (ПК) и сопоставить их с кассами в ППС (Id ПК с Id кассы в ППС).
2. Создать (если не создан) общего кассира для мобильных точек продаж (если не сопоставяться имена кассиров в ПК с кассирами в ППС).
3. В личном кабинете настоящей цифровой создать в общих настройках API ключ для возможности слать http запросы.
4. Заполнить данными из пунктов 1; 2; 3; файл AppConfig.json.
5. Дополнительно заполнить поле с базовым URI (базовый адрес настоящей цифровой https://cabinet.webkassa.by, необходимо уточнить).
6. Похвалить себя за проделанную работу.

	• Кнопка "Импорт из личного кабинета" производит импорт продаж из личного кабинета настоящей цифровой для выбранной программной кассы 
в выпадающем списке "Рег. номер программной кассы" согласно выбранной даты. Выбор даты для иморта производиться слева от кнопки "Импорт из 
личного кабинета". ВАЖНО! {дата после импорта или с течением суток автоматически НЕ ПЕРЕКЛЮЧАЕТСЯ}
	• Галочка "общий импорт" отключает выпадающий список и позволяет осуществить импорт сразу со всех программных касс для выбранной даты.
	• Кнопка "Разрешить автоимпорт" позволяет осуществлять автоимпорт для выбранной программной кассы (или для всех касс, если активна галочка 
"общий импорт") согласно времени слева от кнопки "Разрешить автоимпорт". При этом автоимпорт производиться согласно текущей даты (выбирать дату 
слева от кнопки "Импорт из личного кабинета" НЕ НУЖНО!). Однако при изменеии времени автоимпрота следует выключить(включить), а затем включить(выключить) 
автоимпорт, что бы применить изменеия времени автоимпорта.
	• Кнопка "Создать файл экспорта из ППС" предназначена для формирования файла экспорта каталога товаров из базы данных ППС. В дальнейшем этот файл 
используется для импорта каталога товаров в личном кабинете настоящей цифровой (импорт/экспорт из файла в товарном учете личного кабинета).
